Para poder importar las librerías necesarias y que funcione
correctamente el programa, debe seguir los siguientes pasos:

1. Importar el proyecto a su workspace
2. Asegurarse de que usa la version jre 1.8.0
3. Hacer clic derecho en la carpeta del proyecto (en el workspace)
4. BuildPath/Configure Build Path/Libreries
5. Seleccionarlas todas y presionar "Remove"
5. Ahora debe importarlas
6. Presionar Add External JARs/ProyectoFinal/lib
7. Seleccionarlas todas y presionar "Abrir" y luego "Ok"
8. Ya el proyecto está listo para compilar
